

## Requirements

This project requires the following packages:

Click==8.1.3
Flask==2.2.3
itsdangerous==2.1.2
Jinja2==3.1.2
MarkupSafe==2.1.2
Werkzeug==2.2.3

## Usage

To run this project, simply run the following commands:

```shell
$ python main.py [repo_owner] [repo_name]
```

This will generate a README.md file for the specified repository.
