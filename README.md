

# README

This repository contains a program written in Python, which generates a README.md file with requirements and usage instructions for the code.

## Requirements 

The following packages are needed to run this program:

* click==8.1.3
* Flask==2.2.3
* itsdangerous==2.1.2
* Jinja2==3.1.2
* MarkupSafe==2.1.2
* Werkzeug==2.2.3

## Usage

In order to use the program, use the following commands:

```
python main.py <repo_owner> <repo_name>
```

This will create a README.md file in the GitHub repository of the specified owner and name.
